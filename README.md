## ☺️ Some Tips

1. Install the Gatsby CLI (⭐️ npm 10)

   ```sh
   npm install -g gatsby-cli
   ```

2. Install and use Node (⭐️ 18 or 20) using [NVM](https://github.com/nvm-sh/nvm)

   ```sh
   nvm install --force
   ```

3. Build using Gatsby

   ```sh
   gatsby build
   ```

4. Start the development server (⭐️ local one)

   ```sh
   npm start
   ```

5. Push to production

   ```sh
   npm run deploy
   ```
